1.2. Proof Techniques

1.2.1. Proof by contradiction

Example: Use proof by contracdition to prove 2\sqrt{2}2​ is a irrational number:

  • If 2\sqrt{2}2​ were rational, then it can be written as 2=xy\sqrt{2}= \frac{x}{y}2​=yx​ where x,y∈Qx,y\in Qx,y∈Q and x,y≠0x,y \ne 0x,y​=0
  • By repeatedly cancelling common factors, we can make sure that x,yx,yx,y have no common factors so they are not both even.
  • Since 2=x2y22=\frac{x^2}{y^2}2=y2x2​, so x2x^2x2 is even, then xxx is even.
  • Let x=2wx = 2wx=2w, w∈Nw \in Nw∈N.
  • Then x2=4w2x^2 =4w^2x2=4w2 so4w2=2y24w^2 =2y^24w2=2y2, y2=2w2y^2 =2w^2y2=2w2 so y2y^2y2 is even then so is yyy.
  • This contradicts the fact that x and y are not both even, so 2 being rational, must have been wrong.

1.2.2. Proof by induction

Normal steps:

  1. Prove the first number holds the property.
  2. Prove that if n=mn = mn=m holds the property, then so is n=m+1n=m+1n=m+1.

Examples:

  • For every natural number nnn, 2n+2+32n+12^{n+2} + 3^{2n+1}2n+2+32n+1 is divisible by 777
